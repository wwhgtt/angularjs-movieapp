## Install 

Install local and use the API from http://movieapp-sitepointdemos.rhcloud.com/

    sudo apt-get install nodejs

## Run node

Install packages from `packages.json`.

    npm install

# Run the for local server.

    grunt connect

Browse to `http://localhost:3000`
